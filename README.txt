Οργανισμοί και οργανογράμματα
=============================

Ως «οργανισμό» ορίζουμε τη διάρθρωση ενός φορέα σε επιμέρους μονάδες οι οποίες
ονομάζονται «οργανικές μονάδες» οι οποίες αντανακλούν σε φυσικές δομές που είναι
γνωστές ως «διευθύνσεις», «τμήματα», «γραφεία» κλπ. Πράγματι, οποιοσδήποτε φρέας
απασχολεί ικανό αριθμό ατόμων δεν νοείται να λειτουργεί χωρίς να υφίσταται  κάποιου
είδους οργανωτική δομή που να περιγράφει με σαφήνεια τις λειτουργίες που επιτελεί
ο φορέας και αυτή ακριβώς η περιγραφή ονομάζεται οργανισμός ή, ακριβέστερα, οργανισμός
εσωτερικής λειτουργίας, ή οργανισμός εσωτερικής υπηρεσίας. Για τις ανάγκες της παρούσης
υιοθετούμε τον όρο «οργανισμός εσωτερικής υπηρεσίας» (ΟΕΥ) όπως είθισται να ονομάζεται
ο οργανισμός του Δήμου Θεσσαλονίκης (ΔΘ).

Ο οργανισμός ενός μεγάλου φορέα όπως είναι ο ΔΘ πρέπει να είναι δομημένος ιεραρχικά,
πράγμα που σημαίνει ότι ότι οι διάφορες οργανικές μονάδες δεν είναι ανεξάρτητες
μεταξύ τους, αλλά είναι καταρτισμένες σε κάποιου είδους δενδροειδή δομή στην κορυφή
της οποίας βρίσκεται η Διοίκηση του φορέα, ενώ τις τελευταίες βαθμίδες βρίσκονται
μικρότερες οργανικές μονάδες οι οποίες επιτελούν συγκεκριμένες λειτουργίες δια των
υπαλλήλων που εργάζονατι σε αυτές.

Όπως ήδη αναφέραμε, στην κορυφή της πυραμίδας του οργανισμού ενός μεγάλου φορέα
βρίσκεται η Διοίκηση του φορέα, όπου όμως η έννοια της Διοίκησης δεν είναι σαφώς
καθορισμένη με αποτέλεσμα το τοπίο στην περιοχή αυτή του ΟΕΥ να είναι ομιχλώδες.
Αντίθετα, όσο κατεβαίνουμε τις βαθμίδες του ΟΕΥ η διάρθρωση των οργανικών μονάδων
γίνεται ολοένα και πιο ακριβής, με τις τελευταίες μονάδες της κλίμακας να αποτελούνται
από συγκεκριμένο αριθμό υπαλλήλων οι οποίοι επιτελούν σαφώς καθορισμένες εργασίες
και ως εκ τούτου πρέπει να διαθέτουν σαφώς καθορισμένα προσόντα. Με άλλα λόγια,
η top down περιγραφή ενός οργανισμού δίνει μια γρήγορη αλλά σχετικά ασαφή περιγραφή
των διεργασιών που επιτελεί ο φορέας, ενώ η bottom up περιγραφή του οργανισμού
πρέπει να είναι απολύτως ακριβής και να περιγράφει με σαφήνεια τόσο τις διεργασίες
που καλείται να επιτελέσει η κάθε οργανική μονάδα, όσο και τα άτομα που στελεχώνουν
τις μονάδες αυτές.
